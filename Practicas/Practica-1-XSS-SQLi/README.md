# Práctica Tema 1 - XSS & SQLi
**Alejandro Mamán | INSO3A**

---

## Descripción

Práctica de seguridad web compuesta de dos partes:

**Parte 1 — Herramienta XSS automatizada**  
Script en Python para la detección y explotación de vulnerabilidades XSS 
sobre los 8 primeros niveles de Web for Pentesters (PentesterLab).

Funcionalidades implementadas:
1. Descubrimiento de entradas (GET, POST, path injection)
2. Verificación de reflexión
3. Verificación de persistencia (Stored XSS)
4. Análisis de contexto
5. Detección de filtros
6. Técnicas de evasión
7. Generación de payloads
8. Salida de resultados

**Parte 2 — SQLi → RCE sobre DVWA**  
Proceso completo de auditoría partiendo de SQL Injection hasta lograr 
ejecución remota de comandos (RCE) mediante User Defined Functions (UDF) en MySQL.

## Entorno

- Kali Linux
